# 1. 向量空间
这回要重学小学的四则运算了

**加法和数乘：** 
$$(x,y)\in X \times X \to (x+y) \in X , ~~ (\alpha,x)\in \mathbb{K} \times X \to \alpha x \in X$$

这两个就是加法和乘法。
其中 $\mathbb{K}$ 是数域 $\mathbb{R} ~ or ~ \mathbb{C}$

然后要有一个0元素，和任何元素相加还等于那个元素。
每个元素都还得有一个负元素，保证和它相加为0
（这就形成了一个阿贝尔群，名字很高端。）

然后再有个元素1去搞定乘法部分，就是向量空间了。

不过和我们过去四则运算不同的是，这里的x,y并不一定是个数，而可以是别的什么奇怪的东西。不过 $\alpha$ 还是个数的，所以那里强调叫数乘。

**子空间：** 子空间可以是X的一个子集，也可以是X的子集A张成的子空间，就是A中所有向量的线性组合（啊我想起了高等数值分析的支配。）

A的向量张成的子空间记作 
$$Span ~~ A$$

## 1.1. Hamel 基

X是向量空间，X的Hamel 基是指任何一个由向量 $e_i\in X$ 组成的向量族 $(e_i)_{i\in I}$ 满足：
- 向量族的元线性无关。
- 对任意给定的向量 $x\in X$ ，存在族 $(e_i)_{i\in I}$ 的有限子族 $(e_j)_{j\in J(x)}$ 和数 $x_j \in \mathbb{K}, j\in J(x)$ 使得 $x = \sum_{j\in J(x)}x_je_j$

啊这，和我们线性代数学过的正交基好像啊。第一条简直一模一样，第二条就是所有的向量都可以用这组基的线性组合表示。

在任何向量空间中，Hamel基均是存在的，可以由选择引理导出，不过这个推导过程对我而言就不重要了。

**基数:** 如果向量空间X存在有限的或无限的Hamel基，则X为有限维或相同的无限维空间，其维数就是任意一个Hamel 基的基数，记为 $dim X$

所以向量空间有几维，就有多少种Hamel基？

# 2. 赋范向量空间
顾名思义就是在向量空间的基础上添加了范数。

设 X 是 $\mathbb{K}$ 上的向量空间，其中 $\mathbb{K=R}$ 或 $\mathbb{K=C}$ ，设映射 $||\cdot||: X\to R$ 满足
- $\forall x \in X,||x||\ge 0,当且仅当 x =0 时 ||x|| = 0$
- $\forall \alpha \in \mathbb K,~ x\in X,~ ||\alpha x|| = |\alpha|~||x||$
- $\forall x,y\in X, ||x+y|| \le ||x|| + ||y||$

则称其为X上的范数。偶对 $(X,||\cdot||)$ 则是赋范向量空间。其中X是向量空间， $||\cdot||$ 是范数

可知 

$$|~||x||-||y||~|\le ||x-y||$$

$$||\mathop{\sum}\limits_{i=1}^n x_i||\le \mathop{\sum}\limits_{i=1}^n||x_i||$$

**赋范向量空间的距离：** 设 $(n,||\cdot||)$ 为赋范向量空间，映射 $d:X \times X \to \mathbb{R}$ 定义为对任何 $x,y\in X,~d(x,y) = ||x-y||$ 则d是X的距离

和以前观念里的距离概念差不多

**常见范数定义**
X 是 $\mathbb{K}$ 上的有限维向量空间，其中 $\mathbb{K=R}$ 或 $\mathbb{K=C}$
- 对每个扩充实数 $1\le p\le\infty$ 定义映射 $||\cdot||_p$ 为
$$x = \mathop{\sum}\limits_{i=1}^n x_ie_i \in X \to ||x||_p = (\mathop{\sum}\limits_{i=1}^n |x_i|^p)^{\frac{1}{p}}, 1 \le p < \infty, ||x||_{\infty} = \mathop{max}_{1\le i \le n} |x_i|,p=\infty$$ ，其中p=2时简记为 $|\cdot|:=||\cdot||_2$

- 对这些范数，空间 $(X,||\cdot||_p)$ 是可分的

回顾以下啥叫可分，应该是说这个拓扑空间有一个可数的稠密子集。

有限维赋范向量空间就是可分的

## 2.1. 商空间

X 还是 $\mathbb{K}$ 上的有限维向量空间，其中 $\mathbb{K=R}$ 或 $\mathbb{K=C}$ ，Z 是X的子空间，则有

$$x\sim y  当且仅当 (x-y)\in Z$$

是一个X上的等价关系

用 $[x] = \{y\in X;(x-y)\in Z\} = \{(x-z)\in X;z \in Z\} \subset \mathcal{P} (X)$ 记x模这个关系的等价类。

最开始的基本定义里面提到过商集 $X/\mathcal{R}$，是由 $x$ 的元素模 $\mathcal{R}$ 的等价类全体组成的 $\mathcal{P}(X)$ 的子集。其中那个/读模
在这里，这个商集就应该是这些等价类组成的集合 $X/Z$

在这个商集上定义加法和数乘，$[x]+[y] = [x+y], \alpha [x] = [\alpha x]$
那么就变成了 $\mathbb{ K}$ 上的一个向量空间，就是商空间了。其中的零向量为 $[0] = Z$
常常简记为 $[X]:= X/Z$

网上说**在线性代数中，一个向量空间V被一个子空间N的商是将N“坍塌”为零得到的向量空间，所得的空间称为商空间（quotient space）**
这里的子空间N就是Z，它被坍塌为0了所以变成了商空间的零向量。
[这个例子](https://www.zhihu.com/question/52574116/answer/131112183)说的还蛮直观的，商空间就是把那个子空间压缩成了一个点，作为加法的零元，而为啥叫商空间呢，因为把本来的对称结构均分和我们的除法非常相似。

书上也有一些示例，俺是这么理解的

$e_1,e_2,e_3$ 是我们常见的三维空间的基，那么商空间 $\mathbb{R}^3 / Span ~~e_1$ (这个span是由向量张成的空间的意思)就是平行于直线 $Span ~~ e_1$ 的所有直线组成的向量空间。怎么个压缩法嘞？原来咱们是三维向量，朝哪都行，现在的话就只能和 $e_1$ 平行了，就是压缩了。所有的 $e_1$ 都是加法0元，因为你在这个商空间下的所有向量本来就都是和 $e_1$ 平行的，加上 $e_1$ 也不会改变它的位置，只有沿着 $e_2,e_3$ 方向移动才有改变。
所有三维空间被压掉了一维，还有二维


同样如果我们的商空间是 $\mathbb{R}^3 / Span ~~(e_1,e_2)$ ，那我们就压掉了两维，还剩下一维。
这个空间中的所有元素应该是平行于 $Span~~(e_1,e_2)$ 的平面，那么这些平面的区别就只剩下 $e3$ 不同了，所以 $e_1,e_2$ 都是加了没有什么用的东西，就成为了0元。

## 2.2. 商范数
定义映射 $||\cdot || :X/Z \to \mathbb{R} 为 ||[x]|| = \mathop{inf}\limits_{y\in [x]} ||y||_X = \mathop{inf}\limits_{z\in Z} ||x-z||_X$
则其为商空间 X/Z 上的范数，称为商范数。

这里的inf还是下界。所以这个范数就是 [x] (或者说Z) 里面的范数的下界。
