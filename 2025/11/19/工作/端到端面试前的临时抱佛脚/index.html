Diffusion planner е’Ң pluto

рҹҢҠ Diffusion Planner (з”ҹжҲҗејҸ)

  • дјҳеҠҝпјҡеӨҡжЁЎжҖҒпјҲMulti-modalityпјүжҖ§иғҪдёҠйҷҗй«ҳпјҢиғҪи§ЈеҶівҖңе·ҰеҸіз»•иЎҢеҸ–е№іеқҮеҜјиҮҙж’һеўҷвҖқзҡ„й—®йўҳгҖӮ
  • еҠЈеҠҝпјҡж•°жҚ®йңҖжұӮйҮҸжһҒеӨ§пјҢдё”жҺЁзҗҶйҖҹеәҰж…ўпјҢеңЁж•°жҚ®жңүйҷҗж—¶йҡҫд»Ҙж”¶ж•ӣгҖӮ еҪ“еүҚеӣ°еўғпјҡж•°жҚ®е°‘пјҢеҒҡдёҚдәҶdiffusion plannerпјҢеҸӘиғҪеҒҡpluto

рҹӘҗ Pluto (еӣһеҪ’ејҸ + иҫ…еҠ©д»»еҠЎ)

  • е®ҡдҪҚпјҡйҮҸдә§зә§ SOTA ж–№жЎҲгҖӮ
  • ж ёеҝғдјҳеҠҝпјҡеңЁдёҚдҪҝз”ЁйҮҚеһӢз”ҹжҲҗејҸжЁЎеһӢзҡ„еүҚжҸҗдёӢпјҢйҖҡиҝҮзІҫеҝғи®ҫи®Ўзҡ„зҪ‘з»ңжһ¶жһ„е’Ңиҫ…еҠ©д»»еҠЎпјҢз”Ёиҫғе°‘зҡ„ж•°жҚ®иҫҫеҲ°жһҒдҪізҡ„жіӣеҢ–ж•ҲжһңгҖӮ

иҫ…еҠ©д»»еҠЎзӣ‘зқЈ (Auxiliary Supervision)пјҡ

рҹ“җ жҚҹеӨұеҮҪж•° (Loss Function) зҡ„иҝӣеҢ–

жҷ®йҖҡжЁЎд»ҝеӯҰд№  (Behavior Cloning, BC)пјҡ еҸӘзңӢз»“жһңпјҢе®№жҳ“жӯ»и®°зЎ¬иғҢгҖӮ

$$L_{plan} = || \tau_{pred} - \tau_{gt} ||_2$$ (йў„жөӢиҪЁиҝ№дёҺдё“е®¶зңҹеҖјзҡ„ L2 и·қзҰ») PLUTO жһ¶жһ„пјҡ еј•е…ҘеӨҡйҮҚзӣ‘з®ЎпјҢејәеҲ¶зҗҶи§ЈиҜӯд№үгҖӮ

$$L_{total} = L_{plan} + \lambda_1 L_{forecast} + \lambda_2 L_{state} + \dots$$

рҹ§  дҪ•дёәвҖңејәеҲ¶еӯҰд№ иҜӯд№үвҖқпјҹ

  • зӣ®зҡ„пјҡйҳІжӯўжЁЎеһӢиө°жҚ·еҫ„ (Shortcut) жҲ– жӯ»и®°зЎ¬иғҢ (Rote Memorization)гҖӮ
  • жүӢж®өпјҡж•…ж„Ҹи®ҫи®ЎдёҖдәӣеҫҲйҡҫзҡ„вҖңиҖғйўҳвҖқпјҲиҫ…еҠ©д»»еҠЎпјүгҖӮиҝҷдәӣйўҳзӣ®йқ вҖңжҠ„иўӯйҖҹеәҰвҖқжҲ–вҖңиғҢиғҢжҷҜвҖқжҳҜеҒҡдёҚеҮәжқҘзҡ„пјҢеҸӘжңүзңҹжӯЈзҗҶи§ЈдәҶзәўз»ҝзҒҜгҖҒзҗҶи§ЈдәҶеҲ«дәәзҡ„ж„ҸеӣҫпјҢжүҚиғҪеҒҡеҜ№гҖӮ

иҫ…еҠ©д»»еҠЎзӨәдҫӢ

вң… иҖғйўҳдёҖпјҡTraffic Light State PredictionпјҲзәўз»ҝзҒҜзҠ¶жҖҒйў„жөӢпјү

  • еҰӮжһңдёҚиҖғпјҡжЁЎеһӢеҸҜиғҪеҸӘеӯҰеҲ°вҖңеүҚиҪҰеҒңжҲ‘е°ұеҒңвҖқпјҢж №жң¬дёҚзңӢзәўз»ҝзҒҜпјҲеӣ жһңж··ж·ҶпјүгҖӮ
  • ејәеҲ¶ж•ҲжһңпјҡйҖјиҝ«жЁЎеһӢзҡ„ зү№еҫҒжҸҗеҸ–еҷЁпјҲEncoderпјү еҺ»еӣҫеғҸйҮҢеҜ»жүҫзәўз»ҝзҒҜзҡ„зү№еҫҒгҖӮ
  • жңҖз»Ҳз»“жһңпјҡжЁЎеһӢзҡ„вҖңеӨ§и„‘вҖқйҮҢзңҹжӯЈеҢ…еҗ«дәҶ**вҖңзәўз»ҝзҒҜзҠ¶жҖҒвҖқ**зҡ„иҜӯд№үгҖӮеҚідҪҝеүҚиҪҰй—ҜзәўзҒҜпјҢжЁЎеһӢеӣ дёәзңӢеҲ°дәҶзәўзҒҜпјҢиҮӘе·ұд№ҹдјҡеҒңдёӢжқҘгҖӮ

вң… иҖғйўҳдәҢпјҡFuture Trajectory Prediction of AgentsпјҲйў„жөӢд»–иҪҰиҪЁиҝ№пјү

  • йўҳзӣ®пјҡвҖңжЁЎеһӢпјҢдҪ и§үеҫ—е·Ұиҫ№йӮЈиҫҶеҮәз§ҹиҪҰпјҢ3з§’еҗҺдјҡеҮәзҺ°еңЁе“ӘйҮҢпјҹвҖқ
  • ејәеҲ¶ж•Ҳжһңпјҡиҝ«дҪҝжЁЎеһӢеҺ»еӯҰд№  зӨҫдјҡдәӨдә’ (Social Interaction) е’Ң еҚҡејҲйҖ»иҫ‘гҖӮ
  • жңҖз»Ҳз»“жһңпјҡеҪ“жЁЎеһӢеӯҰдјҡдәҶйў„еҲӨеҲ«дәәпјҢе®ғи§„еҲ’иҮӘе·ұзҡ„и·Ҝзәҝж—¶пјҢе°ұдјҡиҮӘз„¶иҖҢз„¶ең°иҝӣиЎҢ**вҖңйҳІеҫЎжҖ§й©ҫй©¶вҖқ**пјҲдҫӢеҰӮпјҡжҲ‘зңӢе®ғиҰҒеҲҮе…ҘпјҢжҲ‘е°ұжҸҗеүҚеҮҸйҖҹпјүгҖӮ

вң… иҖғйўҳдёүпјҡEgo-Status ReconstructionпјҲиҮӘиҪҰзҠ¶жҖҒйҮҚе»әпјү

  • йўҳзӣ®пјҡвҖңжЁЎеһӢпјҢй—ӯдёҠзңјпјҲеҸӘзңӢдёӯй—ҙеұӮзү№еҫҒпјүпјҢдҪ зҢңзҢңдҪ зҺ°еңЁзҡ„йҖҹеәҰе’ҢеҠ йҖҹеәҰжҳҜеӨҡе°‘пјҹвҖқ
  • ејәеҲ¶ж•Ҳжһңпјҡи®©жЁЎеһӢзҡ„ жҪңж„ҸиҜҶ (Latent Space) ж—¶еҲ»дҝқжҢҒеҜ№зү©зҗҶи§„еҫӢзҡ„ж•Ҹж„ҹгҖӮ
  • жңҖз»Ҳз»“жһңпјҡи®©жЁЎеһӢзҗҶи§Ј вҖңжғҜжҖ§вҖқ е’Ң вҖңеҠЁеҠӣеӯҰзәҰжқҹвҖқгҖӮйҒҝе…Қи§„еҲ’еҮәиҝқеҸҚзү©зҗҶе®ҡеҫӢпјҲеҰӮй«ҳйҖҹзӣҙи§’жҖҘиҪ¬ејҜпјүзҡ„иҪЁиҝ№гҖӮ

иҝҷжң¬иҙЁдёҠжҳҜеңЁеҒҡ Representation LearningпјҲиЎЁеҫҒеӯҰд№ пјү зҡ„зәҰжқҹгҖӮ

  • ж ёеҝғжңҜиҜӯпјҡInductive Bias InjectionпјҲжіЁе…ҘеҪ’зәіеҒҸзҪ®пјүгҖӮ
  • жҖ»з»“йҖ»иҫ‘пјҡ йҖҡиҝҮ еӨҡд»»еҠЎеӯҰд№  (Multi-task Learning)пјҢеҖ’йҖјдё»е№ІзҪ‘з»ң (Backbone) еҺ»жҸҗеҸ–еҮәзңҹжӯЈжңүж„Ҹд№үзҡ„й«ҳеұӮиҜӯд№үзү№еҫҒпјҲи§„еҲҷгҖҒж„ҸеӣҫгҖҒзү©зҗҶзәҰжқҹпјүпјҢиҖҢдёҚжҳҜеҺ»жӢҹеҗҲдёҚеҸҜйқ зҡ„еғҸзҙ зә§еҷӘйҹігҖӮжҸҗй«ҳдәҶжЁЎеһӢеңЁйҷҢз”ҹеңәжҷҜдёӢзҡ„жіӣеҢ–иғҪеҠӣгҖӮ

жҖ»з»“

Pluto е°ұжҳҜйҖҡиҝҮи®ҫи®Ўиҫ…еҠ©д»»еҠЎзҡ„ CostпјҢејәеҲ¶жЁЎеһӢеӯҰд№ й«ҳеұӮиҜӯд№үзҠ¶жҖҒйҮҸпјҢд»ҺиҖҢйҒҝе…Қ ShortcutпјҲеӣ жһңж··ж·Ҷ/жғҜжҖ§дҫқиө–пјүгҖӮ

жЁЎд»ҝеӯҰд№ зҡ„дёӨеӨ§й—®йўҳпјҡ

copycatпјҲеӨҚеҲ¶зҢ«зҢ«пјҢе–өпјҒжғҜжҖ§й—®йўҳпјү

жЁЎеһӢеҸӘжҳҜз®ҖеҚ•еӨҚеҲ¶иҝҮеҫҖзҡ„з»ҸйӘҢжқҘйў„жөӢдёӢдёҖеё§гҖӮпјҲжҚҹеӨұеҮҪж•°жІЎжңүеӯҰеҲ°е…ій”®дҝЎжҒҜпјү

жЁЎд»ҝеӯҰд№ еҰӮжһңеңЁиҮӘеҠЁй©ҫй©¶йўҶеҹҹеҰӮжһңеҮәзҺ°copycatпјҢжҷәиғҪдҪ“зҡ„еҠЁдҪңдёҺж—¶й—ҙејәзӣёе…ігҖӮиҝҮеҫҖзҠ¶жҖҒеҸҜз”ұеҪ“еүҚзҠ¶жҖҒеӣһжәҜжҺЁзҗҶгҖӮ

иҪҰиҫҶдҪңдёәе…ёеһӢзҡ„зү©зҗҶе®һдҪ“пјҢе…¶зҠ¶жҖҒеҸ—жғҜжҖ§дёҺиҝҗеҠЁеӯҰж–№зЁӢзәҰжқҹпјҢе…·жңүжһҒејәзҡ„зҠ¶жҖҒиҝһз»ӯжҖ§пјҲState ContinuityпјүгҖӮзҠ¶жҖҒйқһзӘҒеҸҳжҖ§пјҡ йҖҹеәҰгҖҒеҠ йҖҹеәҰзӯүй«ҳйҳ¶зү©зҗҶйҮҸдёҚдјҡеңЁж—¶й—ҙиҪҙдёҠеҸ‘з”ҹзӘҒеҸҳгҖӮеҪ“еүҚж—¶еҲ»зҡ„зҠ¶жҖҒ $S_t$ еҫҖеҫҖеҢ…еҗ«дәҶжһҒејәзҡ„еҺҶеҸІзҠ¶жҖҒ $S_{t-1}$ зҡ„е…ҲйӘҢдҝЎжҒҜгҖӮеҸҜеӣһжәҜжҺЁзҗҶпјҡ з”ұдәҺиҝҷз§Қиҝһз»ӯжҖ§пјҢиҝҮеҫҖзҠ¶жҖҒеҫҖеҫҖеҸҜд»ҘйҖҡиҝҮеҪ“еүҚзҠ¶жҖҒпјҲжҲ–жһҒзҹӯзҡ„ж—¶й—ҙзӘ—еҸЈпјүиҝӣиЎҢеҸҚеҗ‘жҺЁеҜјгҖӮиҝҷз§Қејәе…ҲйӘҢдҪҝеҫ—жЁЎеһӢе®№жҳ“вҖңеҒ·жҮ’вҖқпјҢйҖҡиҝҮжҺЁз®—еҺҶеҸІжғҜжҖ§жқҘйў„жөӢдёӢдёҖжӯҘпјҢиҖҢйқһйҖҡиҝҮж„ҹзҹҘиҫ“е…ҘжқҘеҶізӯ–гҖӮ

пјҲиҪҰжҳҜеҫҲе…ёеһӢзҡ„жЎҲдҫӢпјҢйҖҹеәҰзӯүзҠ¶жҖҒйҮҸдёҚдјҡеңЁж—¶й—ҙиҪҙзӘҒеҸҳпјҢеҸӘеҠ xyпјҢheadingпјҲheadingе…¶е®һжҳҜдёҚжҳҜд№ҹеҸҜд»ҘдёҚеҠ пјҢеҸҜд»ҘйҖҡиҝҮе·®еҲҶйҖ’жҺЁпјҹпјүпјҢе…¶д»–зҠ¶жҖҒйҮҸеҸҜйҖҡиҝҮз§ҜеҲҶеҫ—еҲ°гҖӮпјү

Embedding (еөҢе…Ҙ)пјҡз”ЁзҪ‘з»ңеӯҰд№ дёҖз»„еҸӮж•°пјҢиҝҷдёӘеҸӮж•°иЎЁзӨәжҹҗдәӣжҰӮеҝөгҖӮжҜ”еҰӮembeddingзҡ„е·ҰиҪ¬жҰӮеҝөе°ұжҳҜеӯҰеҮәжқҘзҡ„дёҖдёӘеҗ‘йҮҸ

йҖ»иҫ‘еұӮпјҲQ/K/VпјҢ Query-Key-Valueпјү

    1. Query (жҹҘиҜў) = ж„ҸеӣҫгҖҒзҠ¶жҖҒзӯүдҝЎжҒҜпјҢеҰӮcommandеҗ‘йҮҸе’Ңtrack queryпјҲиҮӘиҪҰдҪҚе§ҝгҖҒиҪЁиҝ№пјҢе…¶д»–иҪҰиҫҶиҪЁиҝ№зӯүпјү
    1. Key (й”®) = дҪҚзҪ®дҝЎжҒҜпјҲPosпјүе’ҢиҜӯд№үзү№еҫҒгҖӮе®ғзҡ„дҪңз”ЁжҳҜз”ЁжқҘиў« Query еҢ№й…Қзҡ„гҖӮ
    1. Value (еҖј) = еҢ…еҗ«дәҶдҪҚзҪ®е’Ңзү№еҫҒгҖӮе®ғзҡ„дҪңз”ЁжҳҜиў«жҸҗеҸ–зҡ„еҶ…е®№гҖӮ

еңЁUniADдёӯпјҢplan_query(Query (Q)) вҖңжҲ‘иҰҒеҺ»е“ӘпјҹвҖқ(еҢ…еҗ«дәҶ Command + еҪ“еүҚзҠ¶жҖҒ + йў„жөӢи¶ӢеҠҝ)

bev_feat(Key (K) & Value (V)) вҖңең°еӣҫдёҠжңүе•ҘпјҹвҖқ(еҢ…еҗ«дәҶ BEV ең°еӣҫзү№еҫҒ + дҪҚзҪ®зј–з Ғ)

key е’Ң value

keyе’Ңvalueе…ұеҗҢз»„жҲҗдәҶдёҖдёӘеӯ—е…ёпјҢж №жҚ®жҜҸдёҖдёӘkeyеҸҜд»ҘжүҫеҲ°еҜ№еә”зҡ„value

еңЁ attn_module (MultiheadAttention) зҡ„еҶ…йғЁпјҢз¬¬дёҖд»¶дәӢе°ұжҳҜеҒҡзәҝжҖ§еҸҳжҚўпјҲLinear Projectionпјүпјҡ

$$K = \text{bev\\_feat} \times W_K$$ $$V = \text{bev\\_feat} \times W_V$$$W_K$ (Key зҡ„жқғйҮҚзҹ©йҳө)пјҡиҝҷжҳҜзҪ‘з»ңеӯҰеҮәжқҘзҡ„еҸӮж•°гҖӮе®ғзҡ„зӣ®зҡ„жҳҜжҠҠеҺҹе§Ӣзү№еҫҒеҸҳжҲҗйҖӮеҗҲвҖңиў«жҗңзҙўвҖқзҡ„ж ·еӯҗпјҲжҜ”еҰӮзӘҒеҮәдҪҚзҪ®дҝЎжҒҜгҖҒзұ»еҲ«дҝЎжҒҜпјүгҖӮ

$W_V$ (Value зҡ„жқғйҮҚзҹ©йҳө)пјҡиҝҷжҳҜеҸҰдёҖз»„еҸӮж•°гҖӮе®ғзҡ„зӣ®зҡ„жҳҜжҠҠеҺҹе§Ӣзү№еҫҒеҸҳжҲҗйҖӮеҗҲвҖңиў«и§„еҲ’дҪҝз”ЁвҖқзҡ„ж ·еӯҗпјҲжҜ”еҰӮзӘҒеҮәйҖҹеәҰгҖҒеҠ йҖҹеәҰгҖҒжӣІзҺҮдҝЎжҒҜпјүгҖӮ е®ғд»¬жәҗиҮӘеҗҢдёҖдёӘ InputпјҢдҪҶйҖҡиҝҮдёӨдёӘдёҚеҗҢзҡ„е…ЁиҝһжҺҘеұӮ ($W_K, W_V$)

еҰӮдҪ•и®Ўз®—еҫ—еҲ°жіЁж„ҸеҠӣпјҹ

жіЁж„ҸеҠӣ: $Attention(Q, K, V) = Softmax(QK^T/\sqrt{d_k})V$

  1. $QK^T$ (зҹ©йҳөд№ҳжі•)пјҡи®Ўз®— Query е’Ң Key зҡ„зӣёдјјеәҰпјҲд№ҹе°ұжҳҜ Attention ScoreпјүгҖӮ
  2. $Softmax$ (еҪ’дёҖеҢ–)пјҡжҠҠеҲҶж•°еҸҳжҲҗжҰӮзҺҮгҖӮ
  3. $\cdot V$ (еҠ жқғжұӮе’Ң)пјҡж №жҚ®жҰӮзҺҮжҸҗеҸ–дҝЎжҒҜгҖӮ

  2. sdc_traj_query (256)
  3. sdc_track_query (256)
  4. navi_embed (256)

  1. LayerNorm еҪ’дёҖеҢ–пјҡ LayerNorm ејәеҲ¶жҠҠиҝҷ 256 дёӘзү№еҫҒжӢүеӣһеҲ°дёҖдёӘзӣёеҜ№ж ҮеҮҶзҡ„иҢғеӣҙпјҲеқҮеҖјдёә0пјҢж–№е·®дёә1йҷ„иҝ‘пјү

  2. nn.ReLU(inplace=True) йқһзәҝжҖ§зҡ„жҝҖжҙ»еҮҪж•° з„¶еҗҺ.max(1)еҫ—еҲ°жңҖеҘҪзҡ„иҪЁиҝ№дҪңдёәи§„еҲ’ж„Ҹеӣҫ

жҲ‘д»¬и®ӯз»ғзҡ„зҪ‘з»ңдёӯйңҖиҰҒи®ӯз»ғеҲ°е“Әдәӣе‘ўпјҹ

  1. иҫ“е…Ҙзҡ„еӣҫеғҸж•°жҚ®гҖҒеҜјиҲӘжҢҮд»ӨзӯүпјҢз»ҸиҝҮзҪ‘з»ңиҪ¬жҚўжҲҗдёҖз»„еҗ‘йҮҸзү№еҫҒгҖӮ
  2. MLPпјҲеӨҡеұӮж„ҹзҹҘжңәпјү FuserпјҡжҠҠвҖңжҲ‘жҳҜи°Ғ(Track)вҖқгҖҒвҖңжҲ‘еҺ»е“Ә(Command)вҖқгҖҒвҖңжңӘжқҘе’Ӣж ·(Motion)вҖқ е…ҲиһҚеҗҲ иө·жқҘпјҢеҸҳжҲҗдёҖдёӘејәжңүеҠӣзҡ„ QueryгҖӮ
  3. ж„Ҹеӣҫе’ҢиҮӘиҪҰзҠ¶жҖҒзү№еҫҒдҪңдёәQпјҢbevзү№еҫҒдјҡи®ӯз»ғеҫ—еҲ°дёӨдёӘжқғйҮҚзҹ©йҳөпјҢдёҖдёӘжқғйҮҚзҹ©йҳөеңЁе’Ңbevзү№еҫҒеҗ‘йҮҸзӣёд№ҳеҗҺдјҡжҠҠ BEV зү№еҫҒиҪ¬жҚўжҲҗйҖӮеҗҲиў« Query жҗңзҙў зҡ„еҪўжҖҒпјҢиҖҢеҸҰдёҖдёӘжқғйҮҚзҹ©йҳөе’Ңbevзү№еҫҒеҗ‘йҮҸзӣёд№ҳеҗҺдјҡжҠҠ BEV зү№еҫҒиҪ¬жҚўжҲҗйҖӮеҗҲ и§„еҲ’дҪҝз”Ё зҡ„еҪўжҖҒгҖӮ
  4. дҪҝз”ЁSOFTMAXе°Ҷеҗ„дёӘжіЁж„ҸеҠӣеҠ жқғ
  5. з”ҹжҲҗиҪЁиҝ№ (Reg Branch)пјҢд№ҹжҳҜдёҖдёӘMLPпјҡжҠҠжҮӮзҺҜеўғзҡ„ж„ҸеӣҫпјҢйҖҡиҝҮе…ЁиҝһжҺҘеұӮпјҢз®—жҲҗиҪЁиҝ№еқҗж ҮгҖӮ
  6. йҖүеҸ–жңҖдҪіиҪЁиҝ№

зү№еҫҒжҸҗеҸ–дёҺжһ„е»ә

  1. Backbone (дё»е№ІзҪ‘з»ң): еӣҫеғҸйҰ–е…Ҳз»ҸиҝҮзұ»дјј ResNet зҡ„зҪ‘з»ңпјҢжҸҗеҸ– 2D еӣҫеғҸзү№еҫҒгҖӮ

  2. BEVFormer: иҝҷжҳҜжңҖе…ій”®зҡ„дёҖжӯҘгҖӮе®ғйҖҡиҝҮ View TransformationпјҲи§Ҷи§’иҪ¬жҚўпјү е°Ҷ 2D еӣҫеғҸзү№еҫҒвҖңжҸҗеҚҮвҖқеҲ° 3D зҡ„ BEVпјҲйёҹзһ°еӣҫпјү з©әй—ҙгҖӮ

  3. е®ғдҪҝз”Ё Attention жңәеҲ¶пјҢжҹҘиҜў BEV зҪ‘ж јдёӯзҡ„жҜҸдёҖдёӘзӮ№еҜ№еә”еӣҫеғҸдёӯзҡ„е“ӘдәӣеғҸзҙ гҖӮ

  4. иҫ“еҮә ($C$): BEV FeaturesгҖӮиҝҷжҳҜдёҖдёӘй«ҳз»ҙзҡ„зү№еҫҒзҪ‘ж јпјҲжҜ”еҰӮ $200 \times 200$ зҡ„зҪ‘ж јпјүпјҢе®ғе°ұеғҸдёҠеёқи§Ҷи§’зҡ„ең°еӣҫпјҢеҢ…еҗ«дәҶиҪҰйҒ“зәҝгҖҒи·ҜжІҝзӯүйқҷжҖҒдҝЎжҒҜпјҢд№ҹйҡҗеҗ«дәҶйҡңзўҚзү©зҡ„дҪҚзҪ®дҝЎжҒҜгҖӮ

BEV FeatureдёҺ Cost Mapзҡ„еҢәеҲ«

Rule-based Cost Map (ж ҮйҮҸеӣҫ): еҪўејҸ: йҖҡеёёжҳҜдёҖдёӘдәҢз»ҙзҹ©йҳөпјҲ$H \times W$пјүгҖӮ еҶ…е®№: жҜҸдёҖдёӘж јеӯҗпјҲGridпјүйҮҢеҸӘжңүдёҖдёӘж•°еӯ—пјҲScalarпјүпјҢд»ЈиЎЁвҖңз»ҸиҝҮиҝҷйҮҢзҡ„д»Јд»·вҖқгҖӮ BEV Feature (й«ҳз»ҙеј йҮҸ): еҪўејҸ: жҳҜдёҖдёӘдёүз»ҙеј йҮҸпјҲ$H \times W \times C$пјүпјҢе…¶дёӯ $C$ жҳҜйҖҡйҒ“ж•°пјҲChannelпјүпјҢйҖҡеёёжҳҜ 256 жҲ–жӣҙеӨҡгҖӮ еҶ…е®№: жҜҸдёҖдёӘж јеӯҗеҜ№еә”зҡ„дёҚеҶҚжҳҜдёҖдёӘж•°еӯ—пјҢиҖҢжҳҜдёҖдёІй•ҝй•ҝзҡ„еҗ‘йҮҸпјҲжҜ”еҰӮ 256 дёӘжө®зӮ№ж•°пјүгҖӮ зү№зӮ№: еҢ…еҗ«й«ҳз»ҙиҜӯд№үдҝЎжҒҜгҖӮ иҝҷдёІж•°еӯ—йҮҢйҡҗеҗ«дәҶвҖңиҝҷжҳҜдёҖжқЎиҷҡзәҝвҖқгҖҒвҖңиҝҷжҳҜж№ҝж»‘и·ҜйқўвҖқгҖҒвҖңиҝҷжҳҜеҚҠдёӘиҪҰеӨҙвҖқзӯүдё°еҜҢдҝЎжҒҜпјҢз”ҡиҮіжҳҜдәәзұ»иҜӯиЁҖйҡҫд»ҘжҸҸиҝ°зҡ„зү№еҫҒгҖӮ дјҳзӮ№: е®ғжҳҜеҸҜеҜјзҡ„гҖӮеҰӮжһңжңҖз»ҲиҪЁиҝ№дёҺдәәзұ»й©ҫй©¶е‘ҳзҡ„иҪЁиҝ№пјҲGround TruthпјүдёҚдёҖиҮҙпјҢGradientпјҲжўҜеәҰпјүеҸҜд»ҘзӣҙжҺҘйҖҡиҝҮ BEV Feature еҸҚеҗ‘дј ж’ӯеӣһеҺ»пјҢиҮӘеҠЁдҝ®жӯЈ Backbone жҸҗеҸ–зү№еҫҒзҡ„ж–№ејҸгҖӮ

жЈҖжөӢдёҺи·ҹиёӘ (TrackFormer)

Object Queries

Object Queries жҳҜдёҖз»„еӣәе®ҡж•°йҮҸзҡ„еҸҜеӯҰд№ еҸӮж•°пјҲLearnable ParametersпјүгҖӮ

еҒҮи®ҫжҲ‘д»¬еңЁжЁЎеһӢдёӯйў„и®ҫдәҶ $N$ дёӘжҹҘиҜўпјҲдҫӢеҰӮ $N=900$пјүпјҢжҜҸдёӘжҹҘиҜўеҗ‘йҮҸзҡ„з»ҙеәҰдёә $d$пјҲдҫӢеҰӮ $d=256$пјүгҖӮйӮЈд№ҲпјҢObject Queries еҲқе§ӢеҢ–дёәдёҖдёӘзҹ©йҳө $Q_{obj} \in \mathbb{R}^{N \times d}$гҖӮ

A. дҪҚзҪ®йғЁеҲҶ (Positional Embedding) вҖ”вҖ” вҖңеңЁе“ӘйҮҢзңӢвҖқ иҝҷжҳҜ Query зҡ„вҖңй”ҡзӮ№вҖқеұһжҖ§гҖӮз»ҸиҝҮи®ӯз»ғпјҢиҝҷ 900 дёӘ Query дјҡеӯҰд№ еҲ°дёҚеҗҢзҡ„з©әй—ҙеҒҸеҘҪгҖӮ

Query #1 еҸҜиғҪдё“жіЁдәҺ BEV з©әй—ҙзҡ„е·ҰдёҠи§’гҖӮ Query #2 еҸҜиғҪдё“жіЁдәҺеҚҒеӯ—и·ҜеҸЈзҡ„дёӯеҝғгҖӮ еӯҰжңҜи§ЈйҮҠпјҡ е®ғжҳҜеҜ№еңәжҷҜиҝӣиЎҢзЁҖз–ҸйҮҮж ·зҡ„з©әй—ҙе…ҲйӘҢпјҲSpatial PriorпјүгҖӮе®ғе‘ҠиҜүзҪ‘з»ңеҺ»е…іжіЁе“ӘдәӣеҢәеҹҹеҸҜиғҪеҮәзҺ°зү©дҪ“гҖӮ

B. еҶ…е®№йғЁеҲҶ (Content Embedding) вҖ”вҖ” вҖңзңӢеҲ°дәҶд»Җд№ҲвҖқ иҝҷжҳҜ Query зҡ„вҖңе®№еҷЁвҖқеұһжҖ§гҖӮеңЁжңӘз»Ҹи§Јз ҒеҷЁеӨ„зҗҶеүҚпјҢе®ғйҖҡеёёеҲқе§ӢеҢ–дёәйӣ¶жҲ–йҡҸжңәеҖјгҖӮ йҡҸзқҖе®ғеңЁ Transformer Decoder еұӮзә§дёӯжөҒеҠЁпјҲLayer by LayerпјүпјҢе®ғдёҚж–ӯең°йҖҡиҝҮ Cross-Attention д»Һ BEV зү№еҫҒдёӯвҖңеҗёеҸ–вҖқдҝЎжҒҜгҖӮ жңҖз»ҲпјҢиҝҷдёӘеҗ‘йҮҸе°Ҷзј–з Ғзӣ®ж Үзү©дҪ“зҡ„зұ»еҲ«пјҲClassпјүе’ҢеӣһеҪ’еҸӮж•°пјҲBounding Box / VelocityпјүгҖӮ
