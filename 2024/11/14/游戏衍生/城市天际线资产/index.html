城市天际线资产制作完全入门教程：建模篇

利用AI和Blender建造城市天际线的资产

利用AI生成底模

可以用Meshy.ai、Rodin AI 或 Tripo AI之类的3d模型生成AI，我此处选用的是Meshy.ai 准备一个比较标准的图片，目标建筑的正视图、侧视图。尽量避免有大树遮挡或强烈透视畸变的照片。

blender的清理

AI 模型的特点是“面数极多、布线极乱、尺寸不对”。这一步是“驯化”它。

导入与检查： 打开 Blender -> File -> Import。按 Z 键切换到“Material Preview（材质预览）”查看贴图是否正常。

修正坐标原点（至关重要！）：

选中模型，按 Tab 进入编辑模式。

选中最底部的所有点，按 Shift + S -> “Cursor to Selected”（游标移到选中项）。

回到物体模式，右键 -> Set Origin -> “Origin to 3D Cursor”（原点移到游标）。

解释： 游戏的地面就是坐标轴的 Z=0 平面。如果原点在建筑腰部，进游戏后建筑就会半截入土。

调整物理尺寸：

CS 的 1 个格子（Cell）= 8 米。

在 Blender 里新建一个 8m x 8m 的立方体做参考。按 S 键缩放你的建筑模型，使其符合现实比例（比如它现实中占地 32米x32米，就应该占满 4x4 个格子）。
